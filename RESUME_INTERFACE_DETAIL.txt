      • Configuration du RowFactory pour détecter le double-clic
      • Appel de handleDetailRecolte() au double-clic
      • Code:
        tableRecoltes.setRowFactory(tv -> {
            TableRow<Recolte> row = new TableRow<Recolte>() { ... };
            row.setOnMouseClicked(event -> {
                if (event.getClickCount() == 2 && !row.isEmpty()) {
                    Recolte recolte = row.getItem();
                    handleDetailRecolte(recolte);
                }
            });
            return row;
        });

   2. Nouvelle méthode: handleDetailRecolte(Recolte recolte)
      • Ouvre l'interface détail dans une nouvelle fenêtre
      • Charge les données de la récolte
      • Définit le callback de retour à la liste
      • Fenêtre modale (bloque la fenêtre principale)
      • Longueur: ~50 lignes

   3. Modification de handleModifier(Recolte r)
      • Avant: Ouvrait un dialogue simple
      • Après: Appelle directement handleDetailRecolte()
      • Utilise maintenant l'interface détail complète

🚀 COMMENT UTILISER LA NOUVELLE INTERFACE
═══════════════════════════════════════════════════════════════════════════════

DÉMARRAGE:
  1. Lancez l'application: RUN_APP.bat → Option 4
  2. Allez dans "🌾 Récoltes"
  3. Vous voyez la liste des récoltes

OUVRIR L'INTERFACE DÉTAIL:

  Option 1 - Double-clic (RECOMMANDÉ):
    • Double-cliquez sur une récolte
    • La fenêtre détail s'ouvre
    • Voir les informations complètes

  Option 2 - Bouton Modifier:
    • Cliquez sur "✏️ Modifier" dans la colonne Actions
    • La fenêtre détail s'ouvre
    • Voir les informations complètes

DANS L'INTERFACE DÉTAIL:

  Consulter les informations:
    • Tous les champs sont préremplis
    • Lisez les données (ID est en lecture seule)
    • Consultez les statistiques en bas

  Modifier une récolte:
    • Modifiez un ou plusieurs champs
    • Cliquez "💾 Enregistrer"
    • Confirmation: "✅ Récolte modifiée avec succès!"
    • Les données sont mises à jour en base
    • La liste se met à jour automatiquement au retour

  Supprimer une récolte:
    • Cliquez "🗑️ Supprimer"
    • Dialogue de confirmation
    • Cliquez "OK" pour confirmer
    • La récolte est supprimée
    • Vous retournez à la liste

  Actualiser les données:
    • Cliquez "🔄 Actualiser"
    • Les données sont rechargées depuis la base
    • Annule toute modification non enregistrée

  Retour à la liste:
    • Cliquez "← Retour" (en haut ou en bas)
    • La fenêtre détail se ferme
    • Vous retournez à la liste des récoltes
    • La liste est automatiquement mise à jour
