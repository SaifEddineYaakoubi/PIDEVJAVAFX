❓ PROBLÈMES COURANTS ET SOLUTIONS
════════════════════════════════════════════════════════════════════════════════

PROBLÈME 1: "Access denied for user 'root'@'localhost'"
SOLUTION:
   - Vérifiez que le mot de passe est correct
   - Utilisez l'option -p correctement
   - Exemple: mysql -u root -p motdepasse votre_base

PROBLÈME 2: "Unknown database 'votre_base'"
SOLUTION:
   - Vérifiez le nom exacte de votre base de données
   - Utilisez: SHOW DATABASES; pour voir toutes les bases

PROBLÈME 3: "Syntax error"
SOLUTION:
   - Vérifiez qu'il n'y a pas d'erreurs de typage
   - Copiez-collez directement depuis le fichier SQL

PROBLÈME 4: "Column 'type_culture' already exists"
SOLUTION:
   - Les colonnes existent déjà
   - Vous pouvez utiliser directement l'application
   - Aucune action nécessaire
