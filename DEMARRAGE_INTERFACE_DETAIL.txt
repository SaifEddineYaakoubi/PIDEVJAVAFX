⚡ 5 MINUTES POUR TESTER LA NOUVELLE INTERFACE
═══════════════════════════════════════════════════════════════════════════════

ÉTAPE 1 - Démarrer l'application (30 secondes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Double-cliquez sur RUN_APP.bat
  2. Sélectionnez: 4 (Compiler et lancer)
  3. Attendez le démarrage (30 secondes)
  4. L'application démarre

ÉTAPE 2 - Naviguer vers les récoltes (10 secondes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Cliquez sur "🌾 Récoltes" dans le sidebar
  2. La liste des récoltes s'affiche

ÉTAPE 3 - Ouvrir l'interface détail (5 secondes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Double-cliquez sur une récolte
  2. ✨ La fenêtre détail s'ouvre!

ÉTAPE 4 - Tester les fonctionnalités (3 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  TEST 1 - Voir les détails (30 secondes):
    • Regardez tous les champs
    • Vérifiez que les données sont affichées
    • Consultez les statistiques en bas

  TEST 2 - Modifier une donnée (1 minute):
    • Changez la Quantité (ex: 6000 → 7000)
    • Cliquez "💾 Enregistrer"
    • Message: "✅ Récolte modifiée avec succès!"
    • Cliquez "← Retour"
    • Vérifiez que la liste est mise à jour

  TEST 3 - Tester les boutons (1 minute):
    • Double-cliquez sur une autre récolte
    • Cliquez "🔄 Actualiser" (recharge les données)
    • Cliquez "← Retour" (ferme la fenêtre)
    • Testez "✏️ Modifier" depuis la liste

🎯 SCÉNARIOS COURANTS
═══════════════════════════════════════════════════════════════════════════════

SCÉNARIO 1: "Je veux voir tous les détails d'une récolte"
  1. Allez dans "🌾 Récoltes"
  2. Double-cliquez sur une récolte
  3. Regardez tous les champs et statistiques
  ✅ Vous voyez tous les détails!

SCÉNARIO 2: "Je veux corriger la quantité"
  1. Double-cliquez sur la récolte
  2. Changez le champ "Quantité"
  3. Cliquez "💾 Enregistrer"
  4. Cliquez "← Retour"
  ✅ La modification est enregistrée!

SCÉNARIO 3: "Je veux supprimer une récolte"
  1. Double-cliquez sur la récolte
  2. Cliquez "🗑️ Supprimer"
  3. Confirmez la suppression (OK)
  ✅ La récolte est supprimée!

SCÉNARIO 4: "Je veux ajouter une note"
  1. Double-cliquez sur la récolte
  2. Descendez jusqu'à "📝 Notes"
  3. Écrivez votre note
  4. Cliquez "💾 Enregistrer" (facultatif)
  ✅ La note est ajoutée!

❓ QUESTIONS FRÉQUENTES
═══════════════════════════════════════════════════════════════════════════════

Q: Comment ouvrir l'interface détail?
R: Double-cliquez sur une récolte dans la liste

Q: Je n'ai pas vu les données se charger?
R: Regardez la barre supérieure - elle dit "✅ Récolte chargée"

Q: Comment enregistrer mes modifications?
R: Cliquez le bouton "💾 Enregistrer"

Q: Que se passe-t-il si je ferme sans enregistrer?
R: Vos modifications ne sont pas sauvegardées

Q: Comment supprimer une récolte?
R: Cliquez "🗑️ Supprimer" et confirmez

Q: Puis-je restaurer une récolte supprimée?
R: Non, la suppression est définitive

Q: Comment voir les rendements associés?
R: Consultez la zone "Statistiques" (version future: liaison complète)

Q: Les notes sont-elles obligatoires?
R: Non, vous pouvez laisser vides

Q: Puis-je modifier l'ID?
R: Non, l'ID est en lecture seule
