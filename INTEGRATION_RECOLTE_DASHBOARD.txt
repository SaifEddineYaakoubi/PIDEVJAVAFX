🔄 FLUX DE NAVIGATION
═══════════════════════════════════════════════════════════════════

Dashboard.fxml (BorderPane principal)
  └─ Sidebar (VBox)
     ├─ btnDashboard (📊)
     ├─ btnRecoltes (🌾) ← CLIQUE ICI
     └─ btnRendements (📈)

  └─ Center (Zone principale)
     └─ Contient la vue actuelle

Quand on clique sur "🌾 Récoltes":
1. DashboardController.loadRecolteView() est appelé
2. FXMLLoader charge Recolte.fxml
3. RecolteController.initialize() charge les récoltes depuis RecolteService
4. Les récoltes s'affichent dans la TableView

  ✅ dateRecolte (Date)
  ✅ qualite (Qualité: A, B, Premium)
  ✅ typeCulture (Type: Blé, Tomate, etc.)
  ✅ localisation (Localisation: Sfax, Tunis, etc.)

🔗 ARCHITECTURE
═══════════════════════════════════════════════════════════════════

Dashboard.fxml
  └─ DashboardController (gère la navigation)
     ├─ btnRecoltes.setOnAction() → loadRecolteView()

     │     └─ RecolteController (gère les récoltes)
     │        └─ RecolteService.getAll() → BDD
     │
     └─ btnRendements.setOnAction() → loadRendementView()
        └─ Charge Rendement.fxml
           └─ RendementController (gère les rendements)
              └─ RendementService.getAll() → BDD
