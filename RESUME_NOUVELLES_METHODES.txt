1️⃣ CALCUL DE PRODUCTIVITÉ
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   public double calculerProductivite(double quantiteTotale, double surfaceExploitee)

   ➤ Calcule automatiquement la productivité
   ➤ Formule: Productivité = Quantité totale / Surface exploitée
   ➤ Résultat en kg/hectare (ou unité/hectare)
   ➤ Inclut la validation des données

   Exemple:
   ┌─────────────────────────────────────────────────────────────────────┐
   │ double prod = rendementService.calculerProductivite(2550.0, 25.5); │
   │ // Résultat: 100.0 kg/ha                                           │
   └─────────────────────────────────────────────────────────────────────┘

2️⃣ CRÉER UN RENDEMENT AUTOMATIQUEMENT
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   public Rendement creerRendementAutomatique(double quantiteTotale,
                                              double surfaceExploitee,
                                              int idRecolte)

   ➤ Crée un objet Rendement avec productivité calculée automatiquement
   ➤ Ne nécessite que les données essentielles
   ➤ Retourne un objet prêt à être inséré en base de données

   Exemple:
   ┌──────────────────────────────────────────────────────────────────────┐
   │ Rendement r = rendementService.creerRendementAutomatique(           │
   │     2550.0,  // quantiteTotale                                       │
   │     25.5,    // surfaceExploitee                                     │
   │     1        // idRecolte                                            │
   │ );                                                                   │
   └──────────────────────────────────────────────────────────────────────┘

3️⃣ AJOUTER UN RENDEMENT AVEC CALCUL AUTOMATIQUE
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   public boolean ajouterRendementAutomatique(double quantiteTotale,
                                               double surfaceExploitee,
                                               int idRecolte)

   ➤ Crée et ajoute un rendement avec productivité calculée automatiquement
   ➤ Raccourci pratique pour creerRendementAutomatique() + add()
   ➤ Retourne true si l'ajout a réussi, false sinon
   ➤ Affiche les messages de succès/erreur

   Exemple:
   ┌──────────────────────────────────────────────────────────────────────┐
   │ boolean success = rendementService.ajouterRendementAutomatique(     │
   │     2550.0,  // quantiteTotale en kg                                │
   │     25.5,    // surfaceExploitee en ha                              │
   │     1        // idRecolte                                            │
   │ );                                                                   │
   │ if (success) {                                                       │
   │     System.out.println("✅ Rendement ajouté!");                      │
   │ }                                                                    │
   └──────────────────────────────────────────────────────────────────────┘

4️⃣ RECALCULER LA PRODUCTIVITÉ
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   public boolean recalculerProductivite(int idRendement,
                                         double quantiteTotale,
                                         double surfaceExploitee)

   ➤ Recalcule la productivité d'un rendement existant
   ➤ Met à jour le rendement en base de données
   ➤ Utile si les données initiales étaient incorrectes
   ➤ Affiche la nouvelle productivité calculée

   Exemple:
   ┌──────────────────────────────────────────────────────────────────────┐
   │ boolean success = rendementService.recalculerProductivite(          │
   │     5,       // idRendement                                          │
   │     3000.0,  // nouvelleQuantite en kg                               │
   │     30.0     // nouvelleSurface en ha                                │
   │ );                                                                   │
   │ // Résultat: Productivité = 3000.0 / 30.0 = 100.0 kg/ha             │
   └──────────────────────────────────────────────────────────────────────┘

5️⃣ STATISTIQUES AGRÉGÉES
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   a) Productivité moyenne pour une récolte

      public double getProductiviteMoyenne(int idRecolte)

      ➤ Retourne la moyenne des productivités
      ➤ Utile pour comparer les performances

      Exemple:
      double moyenne = rendementService.getProductiviteMoyenne(1);
      System.out.println("Moyenne: " + moyenne + " kg/ha");

   ─────────────────────────────────────────────────────────────────────

   b) Productivité totale pour une récolte

      public double getProductiviteTotale(int idRecolte)

      ➤ Retourne la somme des productivités
      ➤ Utile pour les statistiques globales

      Exemple:
      double total = rendementService.getProductiviteTotale(1);
      System.out.println("Total: " + total + " kg");

   ─────────────────────────────────────────────────────────────────────

   c) Quantité totale récoltée pour une récolte

      public double getQuantiteTotalePourRecolte(int idRecolte)

      ➤ Retourne la somme de toutes les quantités
      ➤ Gère les valeurs NULL automatiquement

      Exemple:
      double qte = rendementService.getQuantiteTotalePourRecolte(1);
      System.out.println("Quantité totale: " + qte + " kg");

   ─────────────────────────────────────────────────────────────────────

   d) Surface totale exploitée pour une récolte

      public double getSurfaceTotalePourRecolte(int idRecolte)

      ➤ Retourne la somme de toutes les surfaces
      ➤ Utile pour les calculs de surface

      Exemple:
      double surface = rendementService.getSurfaceTotalePourRecolte(1);
      System.out.println("Surface totale: " + surface + " ha");

6️⃣ RENDEMENTS EXTRÊMES
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   a) Meilleur rendement (productivité maximale)

      public Rendement getRendementMaximum(int idRecolte)

      ➤ Retourne le rendement avec la meilleure productivité
      ➤ Retourne null s'il n'y a aucun rendement

      Exemple:
      Rendement best = rendementService.getRendementMaximum(1);
      if (best != null) {
          System.out.println("Meilleur: " + best.getProductivite() + " kg/ha");
      }

   ─────────────────────────────────────────────────────────────────────

   b) Pire rendement (productivité minimale)

      public Rendement getRendementMinimum(int idRecolte)

      ➤ Retourne le rendement avec la pire productivité
      ➤ Retourne null s'il n'y a aucun rendement

      Exemple:
      Rendement worst = rendementService.getRendementMinimum(1);
      if (worst != null) {
          System.out.println("Pire: " + worst.getProductivite() + " kg/ha");
      }
